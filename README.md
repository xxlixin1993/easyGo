# easyGo
A golang frame

# 代码规范

- 文件名用下划线命名
- 函数、变量等使用驼峰命名
- 常量以K或k开头
- 代码注释使用双斜线加空格，ex：`// 这是一个注释`
- 提交时 先用`go fmt`工具格式化

# 使用的开源项目

- redis: github.com/gomodule/redigo/redis
- grpc: google.golang.org/grpc
- proto: github.com/golang/protobuf/proto
